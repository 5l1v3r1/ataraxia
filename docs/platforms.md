## Supported platforms
Ataraxia Linux has been ported to 14 different CPU architectures:
```
 * amd64        - for 64-bit x86 CPUs
 * i386         - for 32-bit x86 CPUs beginning with classic Intel Pentium
 * arm64        - for 64-bit ARM CPUs
 * armhf        - for 32-bit ARM CPUs beginning at ARMv7-a (hard-float)
 * armel        - for 32-bit ARM CPUs beginning at ARMv5   (soft-float)
 * mips64       - for 64-bit MIPS CPUs beginning at MIPS Release 1 (big-endian)
 * mips64el     - for 64-bit MIPS CPUs beginning at MIPS Release 1 (little-endian)
 * mips         - for 32-bit MIPS CPUs beginning at MIPS Release 1 (big-endian)
 * mipsel       - for 32-bit MIPS CPUs beginning at MIPS Release 1 (little-endian)
 * m68k         - for 32-bit Motorola 68000 CPUs beginning Motorola 68020
 * riscv64      - for 64-bit RISC V CPUs
 * riscv32      - for 32-bit RISC V CPUs
```
